📚 About The Project

This project is a full conversational multi-document RAG application developed using Streamlit, LangChain, FAISS, HuggingFace Embeddings, and Groq API. Users can upload documents, retrieve semantically relevant information, and interact with uploaded content using conversational AI techniques.
